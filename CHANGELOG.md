# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-05

### Added
- Initial release of Communication Growth Skill
- 6-stage prompt pipeline (P1-P6) for comprehensive communication analysis
- 10-category weakness taxonomy covering all major communication flaws
- Python tool suite: input sanitizer, weakness scorer, report generator
- 2 complete example scenarios (workplace + intimate relationship)
- Methodology library with 8 core frameworks (DESC, Broken Record, NVC, BATNA, etc.)
- Pattern mining across multiple conversations
- Growth planning with SMART goals, stage-based training, and emergency protocols
- Install script for one-command setup

### Core Philosophy
- Reject people-pleasing: effectiveness over likability
- Embrace healthy conflict: distinguish destructive vs. constructive confrontation
- Objective rationality: judge words by strategic value, not emotional comfort
- Methodology-driven: every critique comes with concrete rewrite + theoretical basis
